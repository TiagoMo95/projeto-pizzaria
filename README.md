# 🍕 Sistema de Pizzaria (CRUD)

Projeto desenvolvido com o objetivo de praticar desenvolvimento backend utilizando C# e .NET, simulando o gerenciamento de pedidos de uma pizzaria.

## 🚀 Tecnologias utilizadas
- C#
- .NET / ASP.NET Core
- SQL Server (ou outro banco que você usou)
- Entity Framework (se usou)
- Git

## 📌 Funcionalidades
- Cadastro de pedidos
- Listagem de pedidos
- Atualização de pedidos
- Exclusão de pedidos
- Integração com banco de dados
